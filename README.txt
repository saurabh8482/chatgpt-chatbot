# ChatGPT Chatbot Website

## How to Run

1. Install dependencies:
   pip install -r requirements.txt

2. Set your OpenAI API Key:
   Windows:
       set OPENAI_API_KEY=your_api_key_here
   Linux/macOS:
       export OPENAI_API_KEY=your_api_key_here

3. Start the app:
   python app.py

4. Open your browser at:
   http://127.0.0.1:5000
